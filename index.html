Number Plate Recognition using ML

Timeline: July 2023 – May 2024

Problem: Need for an automated system to accurately identify vehicle number plates from images or live feeds for security and traffic management.

My Role & Solution: As a Project Member, I contributed to developing a user-friendly number plate recognition system. We explored and integrated technologies including Optical Character Recognition (OCR), Convolutional Neural Networks (CNN), Haar Cascade Classifiers (for detection), Google Cloud Vision API, and SQLite for data handling. The system was designed to analyze uploaded images and potentially live camera feeds for effective plate detection and character segmentation/recognition.

Key Technologies: Python, CNN, OCR, Haar Cascade, Google Cloud Vision API, SQLite, Character Segmentation techniques.

Impact & Learning: This project advanced my understanding of machine learning pipelines, computer vision techniques, and API integration. It highlighted the potential of ML in real-world applications like security and automation. Contributed to the publication listed below.
